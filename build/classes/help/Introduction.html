Welcome User!

          Welcome To MathematicalNigerian , a set of mathematical tools built specifically for students and researchers, but targeted generally at an audience that need to carry out basic calculations without having to learn strenuous syntax. The software is an high power scientific calculator replete with extensive graphing, calculations and equation solving ability. Employing easily understood syntax, the software is the tool of choice for anyone desiring access to powerful mathematical computing power combined with syntax as close to normal mathematics as possible.You can use it just like a handheld calculator by clicking buttons, or you can use your keyboard to type the mathematical expressions that you want the calculator to evaluate.
           Some of the functionality included in MathematicalNigerian are listed below:

  1. Basic arithmetic including:
    • Trigonometric operations.
    • Logarithms,Exponential functions.
    • Evaluation of algebraic expressions.
    • Variable and Constant creation and their usage in expressions.
    • Permutations, Combinations, Factorials And Formula Storage, amongst others.
  2. 2D Graph plotting and manipulation capability including:
    • Function plotting e.g y = 3x^2+sin(4x);
    • Point plotting, executed by listing out a set of x coordinates, followed by the set of corresponding y coordinates.
  3. Statistical operations on sets of data including amongst others:
    • Sum of a data set.
    • Average of a data set.
    • Median of a data set.
    • Mode of a data set.
    • Standard deviation of a data set.
  4. Matrix operations including:
    • Matrix variable creation.
    • Reduction to triangular form.
    • Determinants.
    • Joining of 2 matrices along rows or along columns.
    • Matrix arithmetic such as addition,subtraction,multiplication and so on.
  5. Solving systems of linear equations with size ranging from small systems to very large ones.
  6. Finding roots of cubic equations of the form x^3+ax+b=0, where a > 0
  7. Finding roots of quadratic equations of the form x^2+ax+b=0.
  8. Finding roots of equations numerically.
  9. Numerical integration and differentiation.
  10. Function expansion as a polynomial within a given range.
  11. Complex Number Manipulation.
